# Smart_Health_Project
Smart Health Monitoring System built with C++, Node.js, JavaScript, and MongoDB for real-time Heart Rate and BP monitoring with emergency WhatsApp alerts and live location sharing.
# 🏥 Smart Health Monitoring System

## Overview

The Smart Health Monitoring System is an IoT-based healthcare project designed to monitor a user's health in real time. The system connects with health monitoring devices and displays vital health parameters such as Heart Rate and Blood Pressure (BP) on a web dashboard.

When an abnormal health condition is detected, the system automatically sends an emergency WhatsApp alert along with the user's live location to predefined contacts, enabling quick assistance.

## Features

* 📊 Real-time Heart Rate  Monitoring
* 🩺 Live Blood Pressure Tracking
* 🌐 Web-based Dashboard
* 🚨 Automatic Emergency Detection
* 📍 Live Location Sharing
* 💬 WhatsApp Alert Notifications
* 🗄️ MongoDB Database Integration
* 🔗 IoT Device Connectivity

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js
* **Database:** MongoDB
* **Hardware/IoT:** C++ (Arduino)
* **Notifications:** WhatsApp Integration

## How It Works

1. Health sensors collect real-time data.
2. The IoT device processes and sends the data to the server.
3. The web dashboard displays the user's health information.
4. If abnormal readings are detected, an emergency alert is triggered.
5. The user's live location is sent via WhatsApp to emergency contacts.
